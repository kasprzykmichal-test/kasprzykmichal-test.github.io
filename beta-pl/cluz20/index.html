Wersja Beta

v5.07.01 (build 2108)

05 Marzec 2021

  • Firmware dostępny do aktualizacji za pomocą aplikacji Object Manager.

Proces aktualizacji i uruchomienie

Co nowego

  • Obsługa nowych obiektów wirtualnych: SunriseSunsetCalendar, PresenceSensor.
  • Obsługa nowych metod dla modułu LED RGB DIN : SwitchR, SwitchG, SwitchB, SwitchW.
  • Dodanie cech odpowiedzialnych za monitorowanie napięcia zasilania CLU - BusVoltage, BusVoltageSensitivity, MaxBusVoltage, MinBusVoltage.
  • Dodanie cech służących do ustawiania adresu serwera DNS - PrimaryDNS / SecondaryDNS.
  • Obsługa Aeotec Nano Switch ZW139-C.
  • Wsparcie dla widoku diagnostycznego OM.
  • Obsługa ROLLER SHUTTER x1 DIN oraz ROLLER SHUTTER FM w wersji 2.1.1.
